## 📁 Rutas importantes
- Cofre de memoria (este): `C:\GitExtensions\GitExtensions.ZimerfeldLFS\ZimerfeldLFS`
- Raíz del proyecto: `C:\GitExtensions\GitExtensions.ZimerfeldLFS`
- `C:\GitExtensions\GitExtensions.ZimerfeldLFS` **es** el repositorio del proyecto [[📦 GitExtensions.ZimerfeldLFS]] (C#, plugin de GitExtensions para Git LFS)
- GitExtensions instalado en: `C:\Program Files\GitExtensions\`
- Plugins de GitExtensions: `C:\Program Files\GitExtensions\Plugins\`
- Configuración de GitExtensions (fuente del historial de repos): `%APPDATA%\GitExtensions\GitExtensions\GitExtensions.settings`
- Configuración del plugin: `%APPDATA%\GitExtensions\ZimerfeldLFS.language.json` / `ZimerfeldLFS.uisettings.json` / `ZimerfeldLFS.debug.log`
- Proyectos hermanos: [[GitExtensions.ZimerfeldTree]] (árbol de branches) y [[GitExtensions.ZimerfeldCommitMsg]] (mensajes de commit), cada uno con su propio cofre
